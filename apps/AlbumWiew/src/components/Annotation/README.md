# AnnotationOverlay

Componente React per l'overlay di annotazioni (disegni, note) sulle immagini dell'album. Da integrare in AlbumViewer per la gestione delle annotazioni sulle foto.

- Placeholder per futura integrazione di librerie di disegno (es. react-canvas-draw, fabric.js)
- Supporta overlay trasparente e rendering disegni
