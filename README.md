# ImageTools Suite

Repository principale della suite `ImageTools`.

## Struttura

- `apps/`: applicazioni finali della suite
- `packages/`: moduli condivisi
- `docs/`: documentazione di suite e documentazione dei tool

## Tool Attuali

- `apps/auto-layout-app`
- `apps/image-party-frame`

## Launcher Windows

- `avvia-progetto.bat`: schermata di scelta del tool
- `avvia-auto-layout.bat`: avvia direttamente Auto Layout
- `avvia-image-party-frame.bat`: avvia direttamente Image Party Frame

## Documentazione

- `docs/00-overview.md`, `docs/01-tech-stack.md`, `docs/02-ui-system.md`
  sono documenti di suite
- `docs/tools/`
  contiene i documenti specifici dei singoli tool
